# gulp-push
# 执行 npm install
安装依赖包

# 执行 gulp push 命令完成代码一键 webpack打包+add+commit+push        
(具体打包命令可以自己配)
# 执行 gulp childbuild 命令完成cd到子目录并代码一键 webpack打包+add+commit+push  
# 执行 gulp push 命令完成代码一键add+commit+push ，不进行打包只推送

# **********注************ 
现在add  和 commit  的gulp命令中src()配置,需要按照你自己的目录结构设置！！！
